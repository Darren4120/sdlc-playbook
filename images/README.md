# SAFe Lean Startup Cycle Image

Please save the SAFe Lean Startup Cycle diagram image as:
**safe-lean-startup-cycle.png**

in the `images/` folder of the website directory.

The image has been configured to display instead of task cards in the 
"1.1 The Learning Cycle - Lean Startup" section.

## Alternative Formats
If you have the image in a different format (jpg, svg), you can rename it or 
update the reference in script.js (line ~290) to match your image filename.
